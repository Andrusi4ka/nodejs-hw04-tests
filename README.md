# HTTP Server on Pure Node.js

## Опис

Базовий HTTP сервер, реалізований з використанням вбудованих модулів
Node.js (`http`, `url`, `querystring`) без сторонніх бібліотек (без
Express).

Сервер обробляє: - GET запити: `/`, `/about`, `/contact` - POST запит:
`/submit` - Помилки: `400`, `404`, `405`, `500`

------------------------------------------------------------------------

## Встановлення

1.  Переконайтесь, що у вас встановлений **Node.js (v18+
    рекомендовано)**.
2.  Клонуйте репозиторій:

``` bash
git clone <your-repository-url>
cd <project-folder>
```

3.  Встановіть залежності (якщо використовуються тести):

``` bash
npm install
```

------------------------------------------------------------------------

## Запуск сервера

### Звичайний запуск

``` bash
node server.mjs
```

Сервер запускається на порту:

    http://localhost:3000

### Запуск із власним портом

``` bash
PORT=5000 node server.mjs
```

------------------------------------------------------------------------

## Доступні маршрути

### GET /

``` bash
curl http://localhost:3000/
```

Відповідь: - 200 OK - Містить: Home / Welcome to the Home Page

------------------------------------------------------------------------

### GET /about

``` bash
curl http://localhost:3000/about
```

Відповідь: - 200 OK - Містить: About / Learn more about us

------------------------------------------------------------------------

### GET /contact

``` bash
curl http://localhost:3000/contact
```

Відповідь: - 200 OK - Містить: Contact / Get in touch

------------------------------------------------------------------------

### POST /submit

``` bash
curl -X POST http://localhost:3000/submit \
  -H "Content-Type: application/x-www-form-urlencoded" \
  -d "name=John&email=john@example.com"
```

Успішна відповідь: - 200 OK - Повертає Name та Email

------------------------------------------------------------------------

### Некоректні дані

``` bash
curl -X POST http://localhost:3000/submit \
  -H "Content-Type: application/x-www-form-urlencoded" \
  -d "name=&email="
```

Відповідь: - 400 Bad Request - Invalid form data

------------------------------------------------------------------------

### 404 Not Found

``` bash
curl http://localhost:3000/unknown
```

------------------------------------------------------------------------

### 405 Method Not Allowed

``` bash
curl -X PUT http://localhost:3000/
```

------------------------------------------------------------------------

### 500 Internal Server Error

Повертається при внутрішній помилці обробки.

------------------------------------------------------------------------

## Заголовки відповіді

-   Content-Type: text/html; charset=utf-8
-   Content-Length
-   X-Content-Type-Options: nosniff

------------------------------------------------------------------------

## Обмеження реалізації

-   Використовуються тільки вбудовані модулі Node.js
-   Немає збереження даних
-   Базова валідація форми (перевірка на непорожні значення)
-   Немає обмеження розміру POST-запиту
-   Немає захисту від XSS
